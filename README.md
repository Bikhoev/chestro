# Chestro

Учёт объектов, замеров, смет и расходов для строителей.

## Стек

- Next.js 14, React 18, TypeScript
- Tailwind CSS
- Локальное хранилище (localStorage)

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Функции

- Объекты и клиенты
- Замеры (штукатурка, плитка, стяжка и др.)
- Смета с экспортом в печать/PDF
- Расходы (остатки материалов, статус оплаты)
- Авансы, счета, заметки
- Экспорт и импорт данных (JSON)
- Локальный вход/регистрация

## Сборка

```bash
npm run build
npm start
```
